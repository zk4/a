# Action
the simplest, yet powerful autocomplete tool for command line with fzf

# how to use 
1. chanage the path below point to your local reposiotry of `action`
2. put ${pwd}/bin/action in your PATH 
``` bash
#!/bin/bash
```

4. You are ready to go
